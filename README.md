## Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Deployment

Build assets with umi that will be generated to `app/public` which served by egg.

And `manifest.json` will be generated to `config`

```bash
$ npm run build
```

Start server

```bash
$ npm start
```
